Iniciar aplicaciones con Windows

En multitud de ocasiones necesitamos que una aplicación se ejecute cada vez que arranquemos el PC. Algunas aplicaciones ya vienen con la opción de "Iniciar con Windows" pero otras no. Para todas aquellas aplicaciones que no disponen de esta opción es posible iniciarlas a la vez que encendemos el PC de forma automática si seguimos los siguientes pasos:

Carpeta de inicio

Lo primero que tenemos que hacer es acceder a la carpeta que contiene los elementos que se ejecutan al inicio. Dependiendo si queremos que se ejecute solo con nuestro usuario o con cualquiera de los usuarios registrados en el equipo accederiamos a una carpeta de inicio u otra como se muestra a continuación:

  1. Presionamos las teclas WINDOWS + R
  2. Escribimos "shell:startup" (sin las comillas) para abrir nuestra carpeta de inicio o "shell:common startup" para la carpeta de inicio común a todos los usuarios.
  3. Presionamos "Aceptar"

Acceso directo para el inicio

Una vez que tengamos abierta la carpeta de inicio podremos copiar el ejecutable que deseamos iniciar con Windows pero esto no es lo más aconsejable. La mejor alternativa es crear un acceso directo al ejecutable que deseamos iniciar con Windows, de esta manera también podremos pasarle argumentos (en caso de que sea necesario) al propio ejecutable. Para crear el acceso directo basta con seguir los siguientes pasos:

  1. Pulsar el botón derecho del ratón > Nuevo > Acceso directo
  2. Se abre un cuadro de diálogo donde debemos indicar la ruta del ejecutable pulsando "Examinar"

  3. (Opcional) Si queremos pasarle argumentos al ejecutable los añadimos detrás de la ruta. Por ejemplo:

    C:\ejecutable.exe argumento1 argumento2

  4. Pulsamos "Siguiente" y el cuadro de diálogo nos ofrece la posibilidad de cambiarle el nombre. Podemos ponerle el que queramos para identificarlo mejor.
  5. Por último damos a "Finalizar" y ya estaría listo

Compatibilidad

Este método es funcional en las versiones de Windows a partir de Windows 8
