Configurar red raspberry pi server

Publicado el mi. 28 diciembre 2016 por Glaucus Soft

Publicado en Raspberry

Tags: raspberry rasbperry pi wireless wifi network

Configuración de conexión wifi en raspberry pi mediante terminal

Cuando un usuario estándar adquiere una placa raspberry pi lo más probable es que la use con una distribución que ya posee entorno de escritorio gráfico (Raspbian con interfaz Pixel, OSMC, Ubuntu Mate, entre otros). Pues bien, en estos sistemas es muy sencillo configurar la conexión a la red mediante wifi debido a que tenemos un asistente gráfico que nos guía.

El problema viene cuando somos usuarios más avanzados y escogemos un sistema operativo que no tiene entorno gráfico de usuario y con el que tenemos que realizar todas las operaciones mediante comandos en el terminal. La solución es aplicar tan solo estos simples pasos cada vez que queramos conectarnos a una red inalámbrica nueva:

Localización del archivo de configuración

Lo primero que tenemos que hacer es localizar el archivo de configuración que almacena todos los datos de las redes inalámbricas configuradas en nuestra raspberry pi. Este archivo es

/etc/wpa_supplicant/wpa_supplicant.conf

Configuración de la red inalámbrica

Una vez abierto dicho archivo de configuración con el editor de texto que más nos guste, las redes se configuran en su interior de la siguiente manera

network={ ssid="NombreDeLaRed" psk="ClaveDeRed" }

De manera que si tenemos una red que se llama RedInalambrica1 y su contraseña es 123456, el archivo de configuración quedaría asi

network={ ssid="RedInalambrica1" psk="123456" }

Aplicar los cambios

Una vez guardados los cambios en dicho fichero de configuración es necesario reiniciar la red, para ello emplearemos los siguientes comandos

sudo ifdown wlan0

sudo ifup wlan0

y listo, ya tendríamos nuestra raspberry pi con entorno de comandos conectada a nuestra red inalámbrica.
